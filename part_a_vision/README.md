# Part A: Vision & Occlusion Engine

This is the completed Vision & Occlusion Engine (Part A) for the Mega-Heracross Route Resilience project (BAH 2026 Problem Statement 4).

## Architecture
- **Model**: SegFormer (MiT-B3) tailored for 12-channel multispectral input (RGB + NIR + SWIR1 + SWIR2 + NDVI + NDWI + SAR VV + SAR VH + Elevation + Slope).
- **Data Augmentation**: Heavy synthetic cloud/shadow injection and Test-Time Augmentation (TTA).
- **Loss Function**: Combined BCE, Dice, Focal, and Active Contour (clDice) loss to preserve thin road topology.
- **Post-processing**: Aleatoric uncertainty-guided morphology to filter disconnected components.

## Outputs
- `outputs/model.onnx`: The production-ready exported ONNX model.
- `outputs/road_mask.npy`: Binary road mask `(512, 512)` generated by the inference pipeline.
- `outputs/meta.json`: Tile metadata matching downstream requirements for Part B (Routing).

## Quickstart

Run the master integration test (validates contract between Part A and Part B):
```bash
python run_all_tests.py
```

Run inference on a single tile with TTA and Morphology:
```bash
python run_part_a.py
```

Export ONNX model:
```bash
python export_onnx.py
```
